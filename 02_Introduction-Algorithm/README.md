# (02) Introduction Algorithm - Resume

1. Algoritma adalah sebuah prosedur komputasi yang didefinisikan dengan baik yang mengambil beberapa nilai sebagai input dan mengambil beberapa nilai sebagai output.

2. Salah satu contoh algoritma adalah sorting (pengurutan). Misalnya dalam keseharian kita saat menerbangkan layang-layang pastinya terdapat tahapan-tahapan yang harus dilakukan secara urut agar menghasilkan output yang kita inginkan dalam konteks ini (layang layang bisa terbang)

3. Flowchart adalah sebuah salah satu cara untuk menuliskan algoritma dalam bentuk diagram. Di dalam diagram tersebut terdapat simbol yang digunakan untuk menggambarkan urutan dan hubungan antar proses secara detail. Misalnya simbol mulai/selesai digambarkan dalam bentuk oval, simbol input/output dalam bentuk jajar genjang, simbol proses dalam bentuk persegi panjang, dll.
